# fflogs_graphql

This project provides tools and scripts for interacting with the FFLogs GraphQL API, focusing on automating data extraction and processing for jobs and pipelines.

## Features

- Query FFLogs data using GraphQL.
- Automate data collection for various jobs.
- Integrate with data pipelines for further analysis.

## Usage

1. Clone the repository.
2. Configure your API credentials.
3. Run the provided scripts to fetch and process FFLogs data.

## Requirements

- Python 3.8+
- `requests`
- `pandas`
